在Windows系統體驗原味vim

Posted on 2021-03-31 Edited on 2021-05-02

前言

我之前為了用vim在電腦灌了ubuntu，但如果要打osu!會很不方便。所以又試了gvim，但怎麼都用不好。去1!的時候遇到了南一中電神羅文謙，用了超簡單的方法就能在windows上用vim，還可以自訂各種主題和背景，這麼香能不用嗎?

先放個成果圖

香

WSL

首先要裝的就是WSL，它可以讓你把Linux作為Windows的子系統。

這邊簡單說下要怎麼裝，雖然網路上應該很多教學了。

啟用「Windows 子系統 Linux 版」選用功能

Windows設定>>應用程式與功能

打開後滑到最底下

會看到

點選程式和功能

接著再點選左邊的開啟或關閉Windows功能

把Windows子系統Linux版 和 虛擬機平台 勾選起來就可以按下確定了

以系統管理員身分開啟 PowerShell

執行以下兩個指令

接著重新啟動你的電腦

將 WSL 2 設定為預設版本

重啟後再次開啟PowerShell，並執行

安裝 Linux 發行版本

這個步驟非常簡單，直接去Microsoft Store搜尋就好

我這邊會使用Ubuntu做介紹

第一次使用可能會需要設定帳號密碼

Windows Terminals

我之所以不太喜歡用Windows的一個原因就是他的終端機太醜了，沒想到微軟也很有自知之明，所以推出了這款Windows Terminals

下載方式也很簡單，去Microsoft Store搜尋並安裝就行了

下載完後，就可以用它來開啟我們的Ubuntu(或其他用到終端介面的東西都可以)

點選下箭頭應該可以看到一些選項

如果沒看到想要的選項的話可以試試看重新啟動

打開之後就可以像使用Ubuntu一樣盡情的打指令

首先當然是去下載 vim

然後就可以使用拉

完結灑花!

資料存放

你可能會好奇，你在Ubuntu存放的檔案會放到哪裡去

預設會放在 C:\Users\'使用者名稱'

'使用者名稱'是你Windows的使用者名稱

所以你也可以在Windows開啟用Ubuntu編輯的檔案

程式碼複製和 .vimrc 配置

你可能會想要複製vim中的程式碼

你只需要在.vimrc檔案中加上最後面幾行(" WSL yank support這一行以下)

然後複製的時候打上 :%y 就可以了(這是一次複製全部程式碼)

Windows Terminal美化

雖然Windows Terminal原本就不錯看了

但還是可以更進一步美化

按下Ctrl + ,開啟設定

它會開啟一個叫setting.json的檔案

裡面就可以自訂非常多東西，包括字體、顏色、背景等等

這邊放上我的配置

請大家不要直接複製貼上!!!

有些東西是不能直接抄的

這邊和大家說一下可以改的地方

"defaults": 先介紹在這下面可以調整的

"backgroundImage" 背景圖片位置

"backgroundImageOpacity" 背景圖片不透明度

"colorScheme" 這是你要自訂的字體顏色主題名稱(下面會提到)

還有一些像是"font size"之類的東西就不贅述了

接下來是"lists"

這下面放的是你的終端機資料 windows terminals應該會自動幫你處理好

不太需要改

最後是"schemes"

這裡就是可以自訂主題的地方

這邊就是可以copy paste的地方了

你可以放很多不同的主題進去

但一次只能用一個

要到上面提到的"colorScheme"填上你想要的scheme的"name"
