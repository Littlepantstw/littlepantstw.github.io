前言

我之前為了用vim在電腦灌了ubuntu，但如果要打osu!會很不方便。所以又試了gvim，但怎麼都用不好。去1!的時候遇到了南一中電神羅文謙，用了超簡單的方法就能在windows上用vim，還可以自訂各種主題和背景，這麼香能不用嗎?

先放個成果圖

香

WSL

首先要裝的就是WSL，它可以讓你把Linux作為Windows的子系統。

把Windows子系統Linux版 和 虛擬機平台 勾選起來就可以按下確定了

Windows Terminals

我之所以不太喜歡用Windows的一個原因就是他的終端機太醜了，沒想到微軟也很有自知之明，所以推出了這款Windows Terminals

下載方式也很簡單，去Microsoft Store搜尋並安裝就行了

下載完後，就可以用它來開啟我們的Ubuntu(或其他用到終端介面的東西都可以)

點選下箭頭應該可以看到一些選項

如果沒看到想要的選項的話可以試試看重新啟動

打開之後就可以像使用Ubuntu一樣盡情的打指令

首先當然是去下載 vim

然後就可以使用拉

完結灑花!

資料存放

你可能會好奇，你在Ubuntu存放的檔案會放到哪裡去

預設會放在 C:\Users\'使用者名稱'

'使用者名稱'是你在第一次開啟Ubuntu時設定的

所以你也可以在Windows開啟用Ubuntu編輯的檔案
