TOI 2021 1! 心得文

Posted on 2021-03-30 Edited on 2021-03-31

前言

聽說下禮拜段考，那我就來打個心🉐文好了? 反正總之就是想記錄一下 原本想打在我的網站上，但我架完之放太久就忘記要怎麼用了😢。

UPD:又學了好久終於學回來了，所以就放上來了

入營考

去年入營考大概是我第一次接觸到競程比賽，從此之後，我就深深愛上了這種可以申請車馬費去台北玩一天的有趣活動。

還記的去年的入營考，我和同學一起迷路~~

以下為回憶片段

陳昱翔 : 詹挹辰你來台北很多次了，那就給你帶路吧。

詹挹辰 : 好啊。

我 : 我們是要去師大還是師大附中啊?

詹挹辰 : 應該是師大附中吧。

我 : 反正師大和師大附中應該在隔壁吧?

(接著我們來到了師大附中)

我 : 為甚麼這麼多女生阿? 我還以為來考的都會是肥宅ㄟ。

詹挹辰 : 對阿而且怎麼都看起來像大學生。

我 : 為甚麼牆壁上貼著多益的海報啊?

詹挹辰 : ...

陳昱翔 : ...

我 : ...

(後來我馬上叫uber，還好有趕上)

今年入營考當然就沒有迷路了(但全國賽的時候有)，但是這次考試我特別緊張，因為要是這次入營考如果沒考好，之後的競程生涯可能會受到很大的阻礙，而且這不到一年來的努力有沒有成果就看這一次了。

比賽結束後，以為自己已經沒希望了，我賽中預估的分數線大概在300，而我只有拿到266分，但看到記分板我才發現好像大家都燒雞了，我這個分數竟然排在第8名。

但和我一起來的同學成績卻不太理想，連比我強很多的詹挹辰都失常。原本想說可以一起進選訓度假。希望明年可以大家一起進選訓。

其實我也不覺得我打競程特別有天分，CF打了這麼久也只能勉強上藍，學習新演算法的速度也十分緩慢。感覺我今年能進選訓運氣真的很好。

選訓營日常

第一次一個人來台北，覺得有點孤單。而且去選訓的電神我也都不太認識，所以我很擔心能不能融入，還好電神們都很nice，不會排斥和菜雞講話。

我一開始不知道有交誼廳，所以在房間邊緣了2, 3天，後來去洗衣服才發現原來大家都在那。之後就開始了每天下來打桌遊或程式的日常。

室友是YJU，我和他都是比較不會聊天的類型?所以在房間裡很少講話，反而是在交誼廳玩桌遊的時候講比較多話。和他玩桌遊真的很有趣，因為他超怪。

原本想說要認真上課，後來發現大家都在自己刷題或vir，所以後來我就到處去看電神都在解甚麼然後跟題。

ZCK直接上台教教授用cdq分治做三維偏序。

不知道甚麼時候開始，只要下課時間就會有人開始用大螢幕播動漫，從輝夜姬一路播到情色漫畫老師。

下課時間有時候還可以去運動，所以曹宸睿就帶了一群人去跑操場，我原本以為有跑就不錯了，結果被嗆說今年是體育低標。

後來借了籃球和排球，越來越多人下來運動，再加上2, 4都會有體育課，所以多了資訊奧林匹克這個梗。

除了體育課，這次還有拼圖課，我們這組很不幸的選到了一幅印象派畫作 星夜，一開始覺得顏色很多應該不會太難，後來才發現這個想法真是大錯特錯，把印象派畫作切成1000片來看真的不是人該做的行為。

後來我們這組拼到心很累，所以索性就把拼圖時間變成了播油歌時間。

因為安麗美特就在飯店對面，兩個禮拜去了兩次，買了兩個資料夾和兩本小說，大家都好油喔peko。

有一天王政祺在交誼廳開播果青，好多人都一起看，結果balbit說出了超多有夠油的發言，我才發現原來他也很油而且是妹控。

最後幾天開始玩狼人殺，每次都是十幾人局，我之前真的都沒有玩過狼人殺，但不知道為甚麼一直被丟水包。我玩好人的時候都在雷，但玩狼的時候好像都贏?難怪後來大家都不相信我。

結訓後還跑去王政祺家附近吃拉麵和玩桌遊，balbit和YJU兩個新竹人把原本可以好好合作的遊戲玩的超心機。

模考　

一模

31/13/9/100 Rk 11 線上5分

開始後我馬上花15分鐘讀題並思考可做性，果斷開始想pD，想到暴力把線段拆到線段樹上的做法，雖然覺得有可能會被卡記憶體，但寫線段樹不會花我多少時間，所以就寫下去了。吃了一堆RE(或MLE)之類的東西後，終於AC。

剩下的題目真的都不太會了，去把pA和pB的部分分拿完之後，看到pC公告發現他是矩陣快速冪。所以果斷開始寫，推出前兩個subtask和矩陣後，剩一個小時。開始猶豫要不要繼續推，但後來還是放棄了，我開始一直在pA, pB中跳來跳去尋找有沒有其他部分分可拿，最後又回去推pC，想到第三個subtask的時候已經來不及了。

賽後填邪惡表格的時候蠻驚訝自己竟然在線上，但這也讓我更加緊張，原本只是想來度假，但現在卻燃起了一點能盡2!的希望。

二模

37/45/25/19 總Rk 16 線下8分

一樣先看完所有題目，發現我都完全沒想法，一樣先去看pD，希望和上次一樣是水題，看了很就後雖然覺得可以建凸包砸資結，但不敢繼續想下去，因為我自己知道我賽中應該打不出來。

所以我開始撈部分分，pD的暴力判斷先撈掉，pA的13分也先撈掉。但現在已經過去兩個小時了，我開始緊張。

pC和pA都有可以暴力DP的部分分，我都用Top Down亂寫，還好最後有撈到。

突然想到我都沒撈pB所以也把可以暴力的部分撈掉，然後發現還有28分可以用小學排容，就直接暴力排下去(我弱，我不會莫比烏斯)。

現在我最簡單的subtask都被我撈完，我決定繼續觀察pA，我把暴力DP的結果印出來慢慢看。

在最後一分鐘，我看出\(n == 1\)的解法，傳上去WA，原來我dubug的輸出沒拔掉，我拔掉之後馬上重傳，結果竟然被上傳間隔擋掉了，不是最後15分鐘不會有上傳限制嗎?我又重複試了幾次，還是沒辦法，這時時間到了，我安慰自己，"沒上2!就沒上吧，畢竟我這場打得這麼爛，應該不會差這12分吧。"

填邪惡表格的時候我傻眼了，雖然早已有在線下的準備，但我距離線上只差8分，如果我個subtask有拿到的話，如果...，可惜已經沒有如果了，心裡突然湧出滿滿的不甘和懊惱，就差這麼一點進2!，就差這麼一點點。

總結

這次選訓認識了非常多形形色色的電神，原本都只能在FB上看到，現在竟然有機會當面講話，真是開心。我從幾乎完全不認識半個人，到能和大家一起快快樂樂的玩桌遊、看動漫、打程式，只花了兩個禮拜。這兩個禮拜，我體驗到了平常在學校絕對不可能體驗到的生活。這兩個禮拜，絕對是我高中三年最難忘的回憶。

雖然最後以線下8分結訓，說沒有不甘心是不可能的，但我已經非常滿足，有比我離線上更近的人，也有比我強上很多模考卻燒雞的人，能進到2!的只有12人這個事實不會改變，我只能期待明年自己努力變強，不要卡在2!線上下這個尷尬的位置。
